## Overview
This project is an interactive **Python + Streamlit dashboard application** designed to visualize and explore environmental data through a simple web interface. The app focuses on transforming raw data into clear, accessible insights using charts and real-time filtering.

## Dataset
The dashboard uses publicly available **Biscayne Bay water quality data**, including measurements such as temperature, salinity, and other environmental indicators. The dataset is retrieved and processed through API calls and structured data pipelines to ensure accuracy and consistency before visualization.

## Features
- Interactive data visualizations
- API-driven data ingestion
- Modular Python architecture for easy extension
- Real-time filtering and exploration
